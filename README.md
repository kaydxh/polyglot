# 🌐 Polyglot — 编程语言快速学习指南集

> **Polyglot**（多语言者）—— 一站式编程语言快速学习文档库，帮助开发者在最短时间内掌握一门新语言并投入实战。

---

## 📖 项目简介

本项目收集了多种编程语言的**快速学习指南**，每份文档遵循统一的结构和教学方法论，让你能够：

- ⚡ **快速上手** — 10 分钟搭建环境，写出第一行代码
- 📋 **语法速查** — 表格化的语法对照，随用随查
- 🧠 **核心概念** — 每个知识点配可运行代码示例，学了就能用
- 🔨 **实战项目** — 3 个渐进式项目（入门→进阶→综合），边学边练
- ⚠️ **避坑指南** — 新手最常犯的错误及解决方案
- 🗺️ **学习路线** — 1 周 / 1 月 / 3 月分阶段学习计划

## 📐 文档结构

每份语言学习指南都包含以下 **9 大章节**：

```
1. 语言简介与定位    — 特点、应用领域、谁在用
2. 环境搭建          — 安装、编辑器、Hello World
3. 语法速查表        — 变量、类型、控制流、函数（表格形式）
4. 核心概念详解      — 5-8 个核心概念，配代码示例
5. 实战项目          — 入门→进阶→综合，3 个可运行项目
6. 常见陷阱与调试    — Top 10 新手错误 + 调试技巧
7. 生态与工具链      — 包管理、测试框架、常用库
8. 学习路线图        — 1 周 / 1 月 / 3 月分阶段计划
9. 速查备忘录        — 一页纸可打印的语法速查卡
```

## 🎯 教学方法论

本项目的文档基于以下高效学习方法（参考 [Codecademy](https://www.codecademy.com/resources/blog/how-to-learn-a-new-programming-language-fast)）：

| 方法 | 说明 |
|------|------|
| **Learn by Doing** | 每个知识点配可直接运行的代码示例 |
| **Build Real Projects** | 通过渐进式实战项目巩固知识 |
| **Compare & Transfer** | 利用已有语言经验加速学习（对比文档） |
| **Focus on Fundamentals** | 先掌握核心概念，再扩展高级特性 |
| **Fail Fast, Debug Often** | 展示常见错误和调试技巧 |

## 🚀 快速开始

1. **选择你要学的语言**，打开对应的 `*-quick-learn.md` 文件
2. **跟着环境搭建章节**，10 分钟内写出 Hello World
3. **浏览语法速查表**，快速了解基本语法
4. **深入核心概念**，每个概念都有代码示例
5. **动手做实战项目**，从入门到综合逐步提升
6. **遇到问题查避坑指南**，少走弯路

> 💡 如果你已经会一门语言，想学另一门，优先阅读**语言对比指南**，利用迁移学习事半功倍！

## 🤝 贡献指南

欢迎为本项目贡献新的语言学习文档！请遵循以下规范：

- 文件命名：`{语言名}-quick-learn.md`（如 `python-quick-learn.md`）
- 对比文档命名：`{语言A}-vs-{语言B}.md`
- 文档结构：遵循上述 9 大章节模板
- 代码示例：必须可直接运行，不省略 import 或上下文
- 内容比例：代码 : 文字 ≈ 6 : 4
